# 🐦 Flappy Bird Game

A classic Flappy Bird game developed in *Python* using *Pygame*. The player controls a bird, avoids obstacles, and aims to achieve the highest score through smooth and engaging gameplay.

## ✨ Features

- Smooth bird movement
- Random pipe generation
- Collision detection
- Score tracking
- Game Over screen
- Restart functionality
- Pixel-art graphics
- Easy keyboard controls

## 🛠️ Technologies Used

- Python
- Pygame

## 🚀 Installation

1. Clone the repository:
```
bash
git clone https://github.com/karishma24-Itsme/pygame-flappy-bird.git
```

2. Navigate to the project folder:
```
bash
cd pygame-flappy-bird
```

3. Install the required dependencies:
```
bash
pip install -r requirements.txt
```

4. Run the game:
```
bash
python Flappy_bird.py

```
## 🎮 Controls

- *Spacebar* – Fly
- Avoid the pipes and survive as long as possible.
